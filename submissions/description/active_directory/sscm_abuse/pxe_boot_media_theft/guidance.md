# Guidance

Provide a step-by-step walkthrough with a screenshot on how you exploited the vulnerability. This will speed up triage time and result in faster rewards. Please include specific details on where you identified the vulnerability, how you identified it, and what actions you were able to perform as a result.
Include screenshots showing the PXE boot request and successful retrieval of the boot image. Show extracted credentials or sensitive data found within the image. Do not use recovered credentials to authenticate to production systems. Attempt to escalate the vulnerability to perform additional actions. If this is possible, provide a full Proof of Concept (PoC).
